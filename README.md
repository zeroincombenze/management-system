Management System

Odoo modules to manage management systems:

* Quality (ISO 9001)
* Environment (ISO 14001)
* Security (ISO 27001, PCI-DSS)
* IT services (ISO 20001, ITIL)
* Health & Safety (OHSAS 18001)
